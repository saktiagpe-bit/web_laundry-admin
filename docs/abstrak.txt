ABSTRAK

Perkembangan teknologi informasi saat ini mendorong transformasi digital pada sektor Usaha Mikro, Kecil, dan Menengah (UMKM), khususnya jasa laundry yang menjadi pilar pemenuhan kebutuhan domestik perkotaan. Sebagian besar UMKM laundry masih menerapkan pencatatan administrasi secara manual yang rentan terhadap kesalahan pencatatan (mencapai 10% error rate) dan menimbulkan asimetri informasi bagi pelanggan mengenai status pengerjaan pakaian mereka. Penelitian ini bertujuan untuk merancang dan membangun sistem informasi pelayanan dan promosi laundry berbasis web bernama "YURE Laundry" yang mengintegrasikan pencatatan kasir, media promosi, dan pelacakan cucian secara waktu nyata (real-time tracking). Sistem ini dikembangkan menggunakan kerangka kerja (framework) Laravel 12 pada sisi backend, dengan infrastruktur basis data cloud PostgreSQL dari Supabase, serta protokol WebSocket untuk pengiriman data dinamis. Pengembangan sistem menerapkan metode rekayasa perangkat lunak Agile dengan model Scrum. Hasil penelitian menunjukkan bahwa digitalisasi sistem berhasil mengoptimalkan efisiensi operasional dengan memotong waktu pemrosesan transaksi dari rata-rata 15 menit menjadi 5 hingga 8 menit, serta menurunkan tingkat kesalahan pencatatan transaksi hingga di bawah 2%. Fitur real-time tracking yang dibangun berbasis koneksi WebSocket Supabase terbukti sangat responsif dalam menyinkronkan data pembaruan status pengerjaan pakaian dari admin ke layar pelanggan secara instan tanpa perlu memuat ulang (refresh) halaman browser, dengan waktu respon (latency) rata-rata berkisar antara 250 hingga 450 milidetik. Sistem ini secara komprehensif berhasil mendigitalisasi manajemen internal UMKM laundry sekaligus meningkatkan keterbukaan informasi bagi pelanggan.

Kata Kunci: Sistem Informasi, Real-Time Tracking, Laravel 12, Supabase, WebSocket, UMKM.

ABSTRACT

The rapid advancement of information technology has driven digital transformation within Micro, Small, and Medium Enterprises (MSMEs), particularly in laundry services, which have become a cornerstone for urban domestic needs. Most laundry MSMEs still rely on manual administrative record-keeping, which is highly prone to human error (reaching a 10% error rate) and creates information asymmetry for customers regarding their laundry processing status. This study aims to design and develop a web-based laundry service and promotion information system named "YURE Laundry" that integrates cashier administration, digital promotion, and real-time laundry status tracking. The system was developed using the Laravel 12 framework on the backend, integrated with Supabase PostgreSQL cloud database infrastructure, and WebSocket protocol for dynamic data transmission. The system development utilized the Agile software engineering methodology with the Scrum model. The results indicate that the system successfully optimized operational efficiency by reducing transaction processing time from an average of 15 minutes to 5 to 8 minutes, and minimizing transaction record error rates to below 2%. The real-time tracking feature, built on Supabase WebSocket connection, proved to be highly responsive in synchronizing laundry status updates from the admin dashboard to the customer's screen instantly without requiring page reloads, with an average latency of 250 to 450 milliseconds. This system comprehensively digitizes the internal management of laundry MSMEs while significantly improving information transparency for customers.

Keywords: Information System, Real-Time Tracking, Laravel 12, Supabase, WebSocket, MSMEs.
